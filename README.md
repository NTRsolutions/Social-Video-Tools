### Introduction
Welcome 

### Prerequisites
You need to have [Node.js and npm]
- Node node/6.9.0

## Installation

```
npm i
```
### NPM commands
Start server [on port 3000]:
```
npm run server
```

Start worker:
```
npm run worker
```

Start app [you need to run server and worker]
```
npm run dev-server
```

Build app [dev]
```
npm run build-dev
```

Build app [prod]
```
npm run build-prod
```
